# 计算机图形学

## 题目

实现一个点点画曲线软件，能显示点坐标和曲线方程（要求能显示点坐标和曲线方程，能选择样条曲线的次数，能画多段曲线）

## 实现的功能

- 绘制不同次数的 ```Bezier 曲线``` 和 ```B 样条曲线``` 

- 添加、删除曲线

- 添加、删除控制点

- 保存曲线信息，从文件加载曲线信息
